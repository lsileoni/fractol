# fractol
Program for displaying fractals on MacOS with real time exploration features.

## Building on MacOS
```
git clone https://github.com/lsileoni/fractol.git fractol
cd fractol
make bonus
./build/bonus/fractol
```

## Built with minilibx
https://github.com/dannywillems/minilibx-mac-osx
